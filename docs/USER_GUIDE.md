## 系统简介

消防安全一张表是一个综合性的校园安全数据展示平台，整合了车辆管理、人员管理、消防设备、安保监控、宿管数据等多个维度的安全信息，为管理层提供全面的安全态势感知和数据决策支持。

### 主要特点
- 📊 **数据可视化**: 直观的图表展示，清晰呈现数据趋势
- 🔐 **权限管理**: 基于角色的访问控制，保障数据安全
- 📱 **响应式设计**: 完美适配桌面和移动设备
- 📥 **数据导出**: 支持CSV格式导出，便于二次分析
- ⚡ **实时更新**: 数据实时同步，确保信息准确

## 快速开始

### 1. 首次访问

#### 注册账号
1. 打开系统登录页面
2. 点击"注册"标签
3. 输入用户名（仅支持字母、数字、下划线）
4. 输入密码（至少6位）
5. 点击"注册"按钮

**重要提示**: 首个注册的用户将自动成为管理员！

#### 登录系统
1. 在登录页面输入用户名和密码
2. 点击"登录"按钮
3. 登录成功后自动跳转到数据总览页面

### 2. 界面导航

#### 桌面端
- **左侧侧边栏**: 主导航菜单，包含所有功能模块
- **顶部导航栏**: 显示用户信息和操作菜单
- **主内容区**: 显示当前页面内容

#### 移动端
- **顶部菜单按钮**: 点击打开侧边栏
- **用户菜单**: 点击右上角头像图标
- **主内容区**: 可滚动查看内容

## 功能详解

### 1. 数据总览

**功能说明**: 展示各模块的关键指标统计，提供快速访问入口。

**主要内容**:
- 今日车辆通行量
- 在校人数统计
- 访客人数
- 消防设备正常率
- 监控在线率
- 宿舍入住率

**操作指南**:
1. 登录后默认进入数据总览页面
2. 查看各项统计数据
3. 点击快速访问卡片进入对应模块

### 2. 车辆管理

**功能说明**: 管理和查询车辆通行记录，分析车流量趋势。

**主要功能**:
- 车辆通行记录查询
- 车流量趋势图表
- 每日通行量对比
- 数据导出（需权限）

**操作步骤**:

#### 查询车辆记录
1. 点击左侧菜单"车辆管理"
2. 默认显示最近7天的数据
3. 查看统计卡片和图表
4. 滚动到底部查看详细记录表格

#### 筛选日期范围
1. 点击页面右上角的日期选择器
2. 选择开始日期和结束日期
3. 系统自动刷新数据

#### 导出数据
1. 确认拥有导出权限（管理员或已授权用户）
2. 点击"导出数据"按钮
3. 系统生成CSV文件并自动下载

**数据说明**:
- **车牌号**: 车辆的车牌号码
- **识别状态**: 车辆识别的状态（正常通行等）
- **站点**: 通行的门禁站点
- **通过时间**: 车辆通过的具体时间

### 3. 消防安全

**功能说明**: 监控消防设备状态，统计设备正常率。

**主要功能**:
- 设备状态监控
- 设备状态分布图
- 设备正常率统计
- 检测记录查询

**操作步骤**:

#### 查看设备状态
1. 点击左侧菜单"消防安全"
2. 查看设备总数、正常数、异常数
3. 查看饼图了解状态分布
4. 查看设备正常率百分比

#### 查询检测记录
1. 使用日期选择器筛选时间范围
2. 查看表格中的详细记录
3. 关注异常设备（红色标记）

#### 导出设备数据
1. 筛选需要导出的时间范围
2. 点击"导出数据"按钮
3. 下载CSV文件

**状态说明**:
- **正常**: 设备运行正常，无需处理
- **异常**: 设备存在问题，需要关注处理

### 4. 人员管理

**功能说明**: 统计校园人员数据，监测人流量变化。

**主要内容**:
- 在校人数统计
- 今日访客数量
- 今日人流量统计

**使用说明**:
1. 点击左侧菜单"人员管理"
2. 查看各项人员统计数据
3. 了解校园人员分布情况

### 5. 安保监控

**功能说明**: 监控安保设备状态，统计安全事件。

**主要内容**:
- 监控设备在线状态
- 监控在线率统计
- 本月案事件统计
- 反诈劝阻记录

**使用说明**:
1. 点击左侧菜单"安保监控"
2. 查看监控设备在线情况
3. 了解安全事件统计
4. 关注反诈劝阻数据

### 6. 宿管数据

**功能说明**: 统计宿舍入住情况，监控学生归宿状态。

**主要内容**:
- 总住宿人数
- 已归宿人数
- 未归宿人数
- 归宿率统计

**使用说明**:
1. 点击左侧菜单"宿管数据"
2. 查看住宿人数统计
3. 关注归宿率指标
4. 了解未归宿人数

### 7. 权限管理（仅管理员）

**功能说明**: 管理系统用户，分配角色和权限。

**主要功能**:
- 查看用户列表
- 修改用户角色
- 授予导出权限
- 取消导出权限

**操作步骤**:

#### 查看用户列表
1. 以管理员身份登录
2. 点击左侧菜单"权限管理"
3. 查看所有注册用户

#### 修改用户角色
1. 在用户列表中找到目标用户
2. 点击角色下拉菜单
3. 选择"管理员"或"普通用户"
4. 系统自动保存修改

#### 管理导出权限
1. 找到需要授权的用户
2. 点击"授权导出"按钮（或"取消导出"）
3. 系统自动更新权限

**注意事项**:
- 无法修改自己的角色
- 至少保留一个管理员账号
- 权限修改立即生效

## 常见问题

### Q1: 忘记密码怎么办？
**A**: 目前系统不支持自助找回密码，请联系管理员通过后台重置密码。

### Q2: 为什么看不到数据？
**A**: 可能的原因：
- 未登录或登录已过期，请重新登录
- 选择的日期范围内没有数据
- 网络连接问题，请检查网络
- 首次使用需要等待系统初始化数据

### Q3: 导出按钮是灰色的，无法点击？
**A**: 这表示您没有导出权限。请联系管理员在"权限管理"页面为您授予导出权限。

### Q4: 图表显示不正常？
**A**: 尝试以下方法：
- 刷新页面
- 清除浏览器缓存
- 更换浏览器（推荐使用Chrome、Firefox、Edge）
- 检查网络连接

### Q5: 移动端如何使用？
**A**: 
- 点击左上角菜单按钮打开导航
- 所有功能与桌面端一致
- 图表会自动适配屏幕大小
- 表格支持横向滚动查看

### Q6: 数据多久更新一次？
**A**: 
- 页面数据在加载时获取最新数据
- 切换日期范围会重新获取数据
- 建议定期刷新页面获取最新信息

### Q7: 可以同时登录多个账号吗？
**A**: 不可以。一个浏览器只能登录一个账号。如需切换账号，请先退出当前账号。

### Q8: 系统支持哪些浏览器？
**A**: 推荐使用以下浏览器的最新版本：
- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Safari (macOS/iOS)

### Q9: 如何联系技术支持？
**A**: 如遇到问题，请联系系统管理员或技术支持团队。

## 使用技巧

### 提高效率的建议

1. **使用快捷访问**: 在数据总览页面，点击快速访问卡片直接进入对应模块

2. **合理设置日期范围**: 
   - 日常查看使用默认的7天范围
   - 月度统计选择整月范围
   - 年度对比选择跨年范围

3. **定期导出数据**: 
   - 建议每周导出一次数据备份
   - 重要数据及时导出保存

4. **关注异常数据**:
   - 消防设备异常（红色标记）
   - 监控设备离线
   - 未归宿人数异常增加

5. **移动端使用**:
   - 适合查看统计数据
   - 快速了解整体情况
   - 详细操作建议使用桌面端

## 数据说明

### 统计周期
- **实时数据**: 车辆通行、人员流动
- **每日统计**: 归宿情况、访客数据
- **累计数据**: 设备总数、在校人数

### 数据来源
- 车辆数据: 来自门禁系统API
- 消防数据: 来自消防设备检测系统
- 人员数据: 来自校园一卡通系统
- 安保数据: 来自监控管理系统
- 宿管数据: 来自宿舍管理系统

### 数据准确性
- 系统数据来源于各子系统
- 数据实时同步更新
- 如发现数据异常，请及时反馈

## 安全提示

1. **保护账号安全**:
   - 不要与他人共享账号
   - 定期修改密码
   - 使用强密码（字母+数字+符号）

2. **数据保密**:
   - 不要将导出的数据随意传播
   - 妥善保管下载的数据文件
   - 离开时记得退出登录

3. **规范使用**:
   - 仅用于工作需要
   - 不要尝试非法访问
   - 遵守数据使用规范
