## 数据库设计

### 表结构

1. **profiles** - 用户配置表
   - id (UUID, PK)
   - username (TEXT, UNIQUE)
   - email (TEXT)
   - role (ENUM: user/admin)
   - permissions (JSONB)
   - created_at, updated_at

2. **vehicle_data** - 车辆数据表
   - id (BIGSERIAL, PK)
   - plate_number (TEXT)
   - recognition_code, recognition_name
   - station_code, station_name
   - pass_time (TIMESTAMPTZ)
   - raw_data (JSONB)

3. **fire_equipment_data** - 消防设备表
   - id (BIGSERIAL, PK)
   - equipment_number (TEXT)
   - check_date (DATE)
   - status (TEXT)
   - location_code, location_name
   - raw_data (JSONB)

4. **personnel_stats** - 人员统计表
5. **security_stats** - 安保统计表
6. **dormitory_stats** - 宿管统计表

### 安全策略
- ✅ 所有表启用RLS
- ✅ 管理员拥有完全访问权限
- ✅ 普通用户只读权限
- ✅ 用户只能查看/修改自己的profile

## 使用说明

### 首次使用
1. 访问系统登录页面
2. 点击"注册"标签
3. 输入用户名和密码（首个用户自动成为管理员）
4. 登录系统
5. 系统自动生成模拟数据

### 管理员操作
1. 登录后访问"权限管理"页面
2. 查看所有用户列表
3. 修改用户角色（管理员/普通用户）
4. 授予或取消导出权限

### 数据查询
1. 选择功能模块（车辆管理、消防安全等）
2. 使用日期选择器筛选时间范围
3. 查看统计数据和图表
4. 查看详细记录表格

### 数据导出
1. 确认拥有导出权限
2. 筛选需要导出的数据
3. 点击"导出数据"按钮
4. 下载CSV文件
